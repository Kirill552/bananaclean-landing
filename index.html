Questions about Banana Clean

What is the Gemini watermark (Nano Banana)?

Nano Banana is Google's image generation model built into Gemini and AI Studio. It adds a small banana logo to the bottom-right corner of every generated image — 48x48 or 96x96 pixels depending on resolution.

How do I remove the Gemini watermark?

Install Banana Clean from the Chrome Web Store. The extension intercepts downloads from Gemini and AI Studio and removes the watermark automatically. Generate an image, click Download, get a clean file.

Is it free?

You get 15 free cleanings — no card, no sign-up. After that, a one-time $4.99 payment unlocks unlimited lifetime access. No subscriptions.

Which browsers are supported?

Chrome, Edge, Opera, and Brave — any Chromium-based browser that supports extensions from the Chrome Web Store.

Does it upload my images to a server?

No. Watermark removal runs entirely in your browser using the Canvas API. Your images never leave your machine. The extension sends only anonymous usage events for product analytics.

Which websites does it work on?

gemini.google.com and aistudio.google.com — both Google services that use the Nano Banana image generator. The extension cleans images when you download via the built-in Download button.

Is it safe?

Images are processed right in your browser and never sent anywhere. For product analytics, the extension sends only anonymous usage events — no image data, no personal information.

What if Google changes the watermark?

Watermark masks are bundled with the extension and updated through the Chrome Web Store. New pattern — new update.

How does Banana Clean remove the watermark technically?

Banana Clean uses reverse alpha blending: original = (watermarked - α × logo) / (1 - α). The extension has pre-computed masks of the Nano Banana logo at 48x48 and 96x96 pixels. When you download an image, it reads the watermark region, applies the inverse formula pixel by pixel using the Canvas API, and restores the original colors. No AI, no inpainting — pure math.

Can I remove the watermark from Google AI Studio images?

Yes. Banana Clean works on both gemini.google.com and aistudio.google.com. Google AI Studio uses the same Nano Banana image generator, so the watermark is identical and removed the same way.

What is the difference between Banana Clean and online watermark removers?

Online tools require you to upload your image to a third-party server. Banana Clean works as a browser extension — it intercepts the download automatically and removes the watermark locally in your browser. No uploads, no waiting, no privacy risk. Processing takes under 100ms vs several seconds for online tools.

Does it work with "Save Image As" or right-click save?

No. Banana Clean intercepts the official Download button in Gemini and AI Studio. Right-click "Save Image As" saves the in-chat preview which may be lower resolution. Use the Download button for the best quality and automatic watermark removal.

Does Banana Clean remove SynthID?

No. Google embeds two types of watermarks: the visible Nano Banana logo in the corner (this is what Banana Clean removes) and SynthID — an invisible watermark baked into pixel data by Google DeepMind. SynthID cannot be removed by any tool without destroying the image quality. Banana Clean only removes the visible logo.

Does it work with Nano Banana Pro images?

Yes. Nano Banana Pro uses the same visible watermark as the standard Nano Banana model. Banana Clean removes it from both. The extension ships with masks for both 48x48 and 96x96 pixel watermarks used across all Nano Banana variants.
