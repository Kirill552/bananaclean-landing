Privacy Policy Banana Clean

Last updated: March 3, 2026.

Banana Clean is a browser extension and the website banana-clean.app. This page describes what data is used by the extension and the website.

What the extension does

Banana Clean removes the watermark from images when you download them on supported Google websites. Image processing happens locally in your browser. Image contents are never sent to our server.

What we do NOT collect

  • We do not upload your images to a server for processing.
  • We do not collect logins, passwords, chat content, or any personal data from your Google account.
  • We do not sell user data to third parties.

What technical data is used

To evaluate product performance, we use minimal product analytics.

  • Anonymous extension installation identifier.
  • Usage events: install, startup, active day, image clean, popup open, toggle state change, PRO interest.
  • Technical request data sent to the analytics server: event date/time, event source, technical IP address and request origin.

How data is used

  • To count cleanings and assess extension stability.
  • To understand which features users actually need.
  • To measure website traffic, install button clicks, and interest in future features.

What is stored locally in the browser

The extension stores technical metrics and settings in chrome.storage.local, for example: enabled/disabled state, cleaning count, active days count, local anonymous installation identifier, and license key.

Data transmission

Analytics events are sent over HTTPS to our server. Images are never sent to the server for processing.

Payment data

Payments are processed by NOWPayments. We do not store your payment details. We only receive a confirmation that payment was completed and generate a license key.
