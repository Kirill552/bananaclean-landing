Manual Gemini watermark remover FAQ

When should I use this instead of the extension?

Use it when the extension removes most of the Gemini watermark but leaves a visible trace on texture, reflections, marble, water, wood, or clothing.

Is this the same as a generic AI watermark remover?

No. The page is tuned as a Banana Clean fallback for Gemini and Nano Banana image watermarks, with manual brush control for the remaining visible area.

Will the page upload my image?

The cleanup run is local. Optional sample upload is separate, visible, and requires a checkbox plus a Send sample click.
