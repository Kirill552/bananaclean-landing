Privacy Policy Banana Clean

Last updated: March 3, 2026.

Banana Clean provides a paid Nano Banana Prompt Guide, a website at banana-clean.app, and a free bonus browser tool for subscribers. This page describes what data is used by the guide website and bonus tool.

What the bonus tool does

Banana Clean includes a free bonus browser tool that enhances AI image workflows. Processing happens locally in your browser. Image contents are never sent to our server.

What we do NOT collect

  • We do not upload your images to a server for processing.
  • We do not collect logins, passwords, chat content, or any personal data from your Google account.
  • We do not sell user data to third parties.

What technical data is used

To evaluate product performance, we use minimal product analytics.

  • Anonymous bonus tool installation identifier.
  • Usage events: install, startup, active day, local processing count, popup open, toggle state change, PRO interest, checkout interest, and support issue signals.
  • Technical request data sent to the analytics server: event date/time, event source and request origin. We may derive approximate country and network/ASN from request IP for debugging regional behavior, checkout issues, and abuse prevention. Raw IP is not stored in the main analytics event record.

How data is used

  • To count local processing events and assess bonus tool stability.
  • To understand which features users actually need.
  • To measure website traffic, install button clicks, and interest in future features.

What is stored locally in the browser

The bonus browser tool stores technical metrics and settings in chrome.storage.local, for example: enabled/disabled state, local processing count, active days count, local anonymous installation identifier, and bonus key.

Data transmission

Analytics events are sent over HTTPS to our server. Images are never sent to the server for processing. If you request support, an image is received only if you manually send it to us.

Payment data

Payments are processed by Square and Telegram Stars. We do not store your card or wallet details. We only receive payment/subscription status needed to generate the PDF access link and bonus key.
